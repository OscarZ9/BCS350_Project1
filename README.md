# BCS350 Project 1 - Website From Scratch

WEBSITE FROM SCRATCH PROJECT

Overview:
Choose a real or fictional business and create a website completely from scratch.
The project is divided into THREE parts:
1. Content
2. Design
3. Interactivity

You will plan and justify your choices before implementation.
