# BCS350 Project 1 - Website From Scratch

## Project Overview
This project is a complete website for a fictional small business named **Harvest Lane Garden Studio**.  
The site was built from scratch using HTML, CSS, and JavaScript and includes core content, visual design decisions, and interactivity.

## Target Audience
- Homeowners (ages 28-60) who want a clean, attractive garden.
- Busy families who need reliable, low-maintenance outdoor solutions.
- Local residents looking for seasonal landscape support.

## Part 1: Content
1. **Business name:** Harvest Lane Garden Studio  
2. **Real or fictional:** Fictional  
3. **Products/services:** Landscape planning, planting design, seasonal cleanups, monthly maintenance  
4. **Target audience:** Homeowners needing practical and attractive landscaping support  
5. **Problem solved:** Many people want nicer outdoor spaces but lack time, planning experience, and maintenance knowledge  
6. **Unique value:** Simple plans, affordable packages, and seasonal follow-up support  
7. **Website pages:** Home, About, Services, Contact  
8. **Mission statement:** "To make beautiful, healthy outdoor spaces simple and accessible for every homeowner."  
9. **Tone:** Professional, approachable, and practical  
10. **Call-to-action:** Book a consultation  
11. **Homepage must show:** Business identity, service summary, value proposition, and clear CTA  
12. **Media:** Optional project photos and icons to build trust and help users visualize results

## Part 2: Design
1. **Overall style:** Clean, natural, and modern small-business style  
2. **Color scheme:** Green and earth tones to represent nature, calm, and trust  
3. **Fonts:** Sans-serif family for readability and modern appearance  
4. **Spacing/layout:** Wide spacing and section blocks to improve scanning and hierarchy  
5. **Visual hierarchy:** Hero headline -> supporting text -> CTA button -> section details  
6. **Consistency:** Repeated color palette, typography, and spacing system across sections  
7. **Icons:** Leaf, water, and garden tools (planned if expanded)  
8. **Emotions to evoke:** Confidence, calm, and optimism  
9. **Accessibility:** Clear contrast, readable font sizes, semantic structure, form labels, keyboard-friendly controls  
10. **Inspiration sources:** Local landscaping websites, modern service business homepages  
11. **Homepage wireframe (text version):**

```
[Header: Business Name + Navigation]
[Hero: Value proposition + CTA button]
[About section]
[Services section]
[Contact form]
[Footer]
```

## Part 3: Interactivity
1. **Interactive elements:** Navigation links, CTA button, contact form  
2. **Navigation approach:** Top navigation with anchor links to each section  
3. **Validation/feedback:** JavaScript checks for required fields and email format; message shown to user  
4. **JavaScript functionality:** Form validation, feedback messages, and form reset on successful submission  
5. **UX improvement:** Prevents invalid form submissions and provides clear immediate feedback

## Information Organization
- Header introduces the brand and navigation.
- Hero states main value quickly.
- About and Services provide trust and details.
- Contact section captures leads.
- Footer closes with brand identity.

## Visual Design Summary
- Light background with green accent colors.
- Consistent spacing and card-like form layout.
- Mobile-responsive adjustments for smaller screens.

## Interaction / Functionality
- Contact form with:
  - Required field checks
  - Email format validation
  - Success/error status messaging

## Technical Overview
- **HTML:** Semantic structure with sections and labeled form fields  
- **CSS:** Custom styling, responsive behavior, spacing system, and accessible focus states  
- **JavaScript:** Client-side validation and user feedback (`script.js`)

## Timeline / Milestones
1. Business concept and content planning  
2. Wireframe and design system decisions  
3. HTML structure implementation  
4. CSS styling and responsive refinements  
5. JavaScript interactivity and form validation  
6. Final testing and deployment to GitHub Pages
